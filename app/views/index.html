International claim capture

Version 1

First version of new international claim capture. Claimant has made no previous claims for Winter Fuel, State Pension, or any other benefit. Therefore the agent has to manually input all data.

Start now

Version 1.1

Example where NI number pulls through some data. With the option to change the data which has been pre-populated if necessary.

NI number: QQ123456C
name: Benjamin Fischer

Start now

Version 2

Research:
Testing version 1 prototype with international claim agents.

Findings:
• Changed the order of questions and placed section names above H1. Also placing ‘knock out’ questions at the beginning
• Some agents missed the hint text- so changed it to inset text
• Took out check your answers because agents weren’t reading it, instead added ‘recap’ pages for important information to read back to customer e.g nino, address etc
• Added ‘preparation’ screen for some groups of questions e.g custody, hospital
